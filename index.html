DCC704 - Arquitetura e Tecnologias de Sistemas Web

A disciplina DCC704 - Arquitetura e Tecnologias de Sistemas Web do DCC/UFRR tem como objetivo apresentar ao aluno
conceitos sobre desenvolvimento Web Front-end e Back-end. Sua carga horária apresenta um total de 60 horas-aula e será
ministrada pelo Prof. Acauan Ribeiro. Para as aulas serão utilizadas exposições e atividades práticas, além de avaliações
individuais e em grupo.

Conteúdo

Dentre os principais temas que serão abordados na disciplina pode-se listar:

  1. Estrutura da Web
    1. Função do cliente e servidor HTTP
    2. Componentes para construir um sistema Web
  2. HTML
    1. Hypertexto
    2. Tag, atributo, Entidades
    3. Estrutura do HTML
  3. CSS
    1. Folha de estilo
    2. Herança e cascata
    3. Seletores, propriedades, funções, media query, at-rules
    4. Definição de layouts e estilos
    5. Animações
    6. Extensões de CSS
  4. XML
  5. Definição
  6. Estrutura
  7. Validação

Além disso, é possível visualizar os detalhes do roteiro das aulas, das ferramentas utilizadas e de alguns links úteis.
