DCC704 - Arquitetura e Tecnologias de Sistemas Web
Logo LM
A disciplina DCC704 - Arquitetura e Tecnologias de Sistemas Web do DCC/UFRR tem como objetivo apresentar ao aluno conceitos sobre desenvolvimento Web Front-end e Back-end. Sua carga horária apresenta um total de 60 horas aula e será ministrada pelo Prof. Acauan Ribeiro. Para as aulas serão utilizadas exposições e atividades práticas, além de avaliações individuais e em grupo.

Conteúdo
Dentre os principais temas que serão abordados na disciplina pode-se listar:
Estrutura da Web
  Função do cliente e servidor HTTP
  Componentes para construir um sistema Web
HTML
  Hypertexto
  Tag, atributo, Entidades
  Estrutura do HTML
CSS
  Folha de estilo
  Herança e cascata
  Seletores, propriedades, funções, media query, at-rules
  Definiação de layouts e estilos
  Animações
  Extenções de CSS
XML
  Definição
  Estrutura
  Validação
